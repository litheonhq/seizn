# 개인정보처리방침

최종 갱신일: 2026-05-02

## 1. 사업자

Seizn (이하 'Seizn' 또는 '당사')은 미국 와이오밍주 유한책임회사인 Litheon LLC가 운영합니다. Litheon LLC는 Seizn 서비스를 통해 처리되는 개인정보의 단독 개인정보처리자입니다.

문의: privacy@seizn.com

## 2. 적용 범위

본 방침은 seizn.com 방문, Seizn Author 가입, engine.seizn.com 사용, 관련 서비스 이용 시 수집되는 개인정보를 다룹니다. 사용자가 직접 연결한 외부 서비스 (BYOK 경로의 Anthropic·Google·OpenAI 등)는 각 사의 개인정보처리방침이 별도로 적용됩니다.

## 3. 수집하는 정보

### 3.1 사용자가 직접 제공하는 정보

- 계정 정보: 이메일, 이름, 비밀번호 해시, 언어 설정.
- 결제·구독: Stripe가 처리하며, 당사는 고객 ID·플랜·상태·청구 메타데이터만 받습니다. 카드 전체 번호는 보관하지 않습니다.
- 작가 콘텐츠: 원고, 세계관 노트, 캐릭 데이터, 씬, 검수 결정 등.
- 문의·지원: support@seizn.com 또는 인앱 폼을 통해 보낸 메시지.

### 3.2 자동으로 수집되는 정보

- 사용 텔레메트리: 페이지 조회·기능 사용·오류·성능 메트릭. 가능한 경우 익명화.
- 기기·접속 정보: IP 주소, 브라우저, OS, 언어.
- 쿠키 및 유사 기술: §10 참조.

### 3.3 제3자로부터 받는 정보

- OAuth 제공자 (Google, GitHub) 로그인 시: 이메일, 프로필명, 아바타 URL.
- 사용자가 연결한 동기화 소스 (Google Docs, Notion, Obsidian): 명시적으로 권한을 부여한 문서·폴더만.

## 4. 개인정보 이용 목적

다음 목적으로만 처리합니다:

- Seizn 서비스 제공·운영 (계약 이행).
- 구독·결제 처리 (계약 이행).
- 서비스 공지·보안 알림·동의 시 제품 업데이트 발송 (동의 또는 정당한 이익).
- 제품 품질 개선·악용 탐지 (정당한 이익).
- 법적 의무 이행 (세무·회계·법령상 요청 대응).

당사는 개인정보를 판매하지 않습니다. 작가 콘텐츠를 파운데이션 모델 학습에 사용하지 않습니다. 광고·마케팅 목적으로 작가 콘텐츠를 제3자와 공유하지 않습니다.

## 5. AI 기능 및 모델 제공자

Seizn은 캐논 추출·충돌 검출·씬 시뮬레이션을 위해 대규모 언어 모델 (LLM)을 사용합니다. 라우팅은 사용자 설정에 따라 다릅니다:

- 기본: Seizn이 선택한 제공자 (현재 Anthropic)로 요청 전송. 응답 전달에 필요한 범위 외에는 콘텐츠를 보관하지 않습니다.
- BYOK (Bring Your Own Key): 사용자가 자신의 API 키를 등록하면, 해당 제공자로 직접 요청이 가서 사용자 계정·과금 관계 하에 처리됩니다. Seizn은 요청을 통과시키되 보관하지 않으며, 제공자의 개인정보처리방침이 적용됩니다.

제공자 데이터 보관 정책은 각 사의 정책을 따릅니다. 활성 제공자 및 보관 기간은 seizn.com/docs/providers에 명시합니다.

## 6. 제3자 공유

다음 경우에만 공유합니다:

- 당사 지시에 따라 처리하는 수탁자: Stripe (결제), Cloudflare (CDN·R2 객체 저장소), Vercel (호스팅), Sentry (오류 모니터링), PostHog 또는 Plausible (개인정보 친화적 분석), Anthropic (AI 추론·기본 라우팅), Google·GitHub (OAuth 로그인 한정).
- 법령상 요구 (법원 명령·소환·합법적 요청).
- 합병·인수 시, 사전 통지 후.

수탁자 목록은 seizn.com/docs/subprocessors에서 확인할 수 있습니다.

## 7. 데이터 저장 위치

작가 콘텐츠는 워크스페이스 단위로 저장 위치를 선택할 수 있습니다:

- 서울 (아시아·태평양 기본값)
- 도쿄
- 프랑크푸르트 (EU GDPR 대응)

계정 메타데이터·결제 기록·운영 로그는 미국에서 Litheon LLC와 수탁자가 처리합니다. 국가 간 이전은 해당 시 표준 계약 조항 (SCC)을 근거로 합니다.

## 8. 보안

- 전송 중 암호화 (TLS 1.2+) 및 저장 시 암호화 (AES-256).
- 접근 통제: 최소 권한·감사 로그.
- BYOK 암호화 시크릿은 API 키와 분리 저장.
- 정기 의존성 취약점 스캔 (런칭 시점 critical 0건).

완벽한 시스템은 없습니다. 개인정보 침해를 인지하면 GDPR·KISA·관련 법령에 따라 영향받는 사용자와 감독 기관에 통지합니다.

## 9. 사용자의 권리

거주 지역에 따라 다음 권리를 행사할 수 있습니다:

- 보유 개인정보 열람.
- 부정확한 정보 정정.
- 작가 콘텐츠 내보내기 (Settings → Export·요청 불필요).
- 계정·관련 데이터 삭제 (Settings → Account → Delete; 확인 후 30일 이내 처리·법적 보관 의무 예외).
- 처리 제한·반대.
- 동의 철회.
- 감독 기관에 신고 (예: 한국 KISA, 일본 PPC, 중국 CAC, EU 데이터 보호 기관, 캘리포니아 CCPA).

인앱에서 행사 불가능한 권리는 privacy@seizn.com으로 문의하시기 바랍니다. 30일 이내 회신합니다.

## 10. 쿠키 및 텔레메트리

세션·CSRF 토큰·언어 설정·동의 시 분석에 한정된 최소 쿠키만 사용합니다. 분석 쿠키는 거부해도 기능 제한이 없습니다. 광고용 제3자 쿠키는 사용하지 않습니다.

## 11. 아동 정보

Seizn은 아동을 대상으로 하지 않습니다. 16세 미만 사용자의 개인정보는 의도적으로 처리하지 않습니다. 미성년자가 계정을 만들었다고 의심되는 경우 privacy@seizn.com으로 알려주시면 계정을 삭제합니다.

## 12. 국가 간 이전

Seizn은 글로벌 서비스로, 미국·한국·일본·독일 등 수탁자가 운영하는 관할에서 데이터를 처리할 수 있습니다. 필요한 경우 SCC, 적정성 결정, 또는 명시적 동의를 근거로 합니다.

## 13. 보관 기간

- 계정·콘텐츠: 계정 삭제 시까지.
- 결제 기록: 세무·회계 의무에 따라 7년.
- 사용 텔레메트리: 식별 가능 형태 13개월, 집계 형태는 그 이후.
- 감사 로그 (캐논 결정·AI 후보 검수): 워크스페이스 수명 동안·요청 시 내보내기 가능.

## 14. 방침 변경

본 방침은 변경될 수 있습니다. 중대한 변경은 효력 발생 30일 전 이메일 및 인앱 배너로 안내합니다. 이전 버전은 seizn.com/legal/privacy/history에서 열람할 수 있습니다.

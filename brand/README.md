# Seizn Brand Assets — Source

**Confirmed**: 2026-04-21

## Concept

Four colored season nodes (Green=Spring, Coral=Summer, Gold=Autumn, Teal=Winter) orbit a central stylized "S" mark. The season cycle encodes Seizn's core semantic: memory as temporal rhythm, not static storage.

## Favicon caveat

At 16×16, the S detail disappears and only the four colored dots remain legible. A simplified favicon variant may be needed:

- Option 1: Keep the full mark; users see colored dots at small sizes (brand remains coherent).
- Option 2: Generate a separate simplified favicon (just the four dots in a circle, no S).

Decision deferred — ship Option 1 first, revisit if feedback says small-size favicon is unclear.

QA note, 2026-04-21: Browser-rendered 16px favicon keeps the four seasonal dots visible and only partially preserves the S detail. STAGE-BA-07 ships Option 1, the full mark, and keeps the simplified-dot variant as a future fallback.

## Dark mode

Mark is designed on transparent background with white-ish S strokes. On `--seizn-ink #0a0a12`, the S may be low-contrast. If visual regression is bad, add a subtle inner glow to the S strokes or increase their opacity.

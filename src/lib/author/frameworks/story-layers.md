# Story Layers (condensed reference)

Distilled from PAI WriteStory `StoryLayers.md` (MIT — Daniel Miessler). See `LICENSE-attribution.md`.

Every scene in long-form fiction operates across seven simultaneous layers. Each exists independently, but they reinforce each other. A scene that advances only one layer is wasted space.

## The Seven Layers

| # | Layer | What it is |
| - | --- | --- |
| 1 | Meaning | The story's philosophical argument. Emerges through character action, never stated. |
| 2 | Character Change | Sacred flaw → crisis → transformation. The narrative engine. |
| 3 | Plot | The causal chain — each event causes the next. |
| 4 | Mystery | Information management. Drives engagement across every genre. |
| 5 | World | Setting that creates pressure and reflects theme. |
| 6 | Relationships | Bonds as miniature stories. The influence character challenges the sacred flaw. |
| 7 | Prose | Sentence-level craft. Voice, rhetoric, anti-cliche discipline. |

## Layer Interaction Rules

1. Every scene advances at least two layers.
2. Character change is primary. The other layers exist to pressure the sacred flaw.
3. Mystery sustains engagement when plot slows.
4. Theme emerges through action — never exposition.
5. World serves story. Every detail eventually matters for character or plot.
6. Relationships create the emotional stakes that make plot events matter.
7. Prose matches the moment — simple for action, rich for emotional peaks.

## Mapping Template

For each major beat, map all seven layers:

```
BEAT: [Name] at [percentage]
MEANING:      What thematic element is present or advanced?
CHARACTER:    Where is the sacred flaw? What pressure?
PLOT:         What event? What causes it? What does it cause?
MYSTERY:      What questions open? Close? Redirect?
WORLD:        What environment detail matters? New revelation?
RELATIONSHIP: Which bonds are tested, deepened, or broken?
PROSE:        What rhetorical emphasis? What voice register?
```
